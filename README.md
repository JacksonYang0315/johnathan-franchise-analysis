# 強納森人力 — 加盟投資分析報告

> 互動式加盟投資分析工具，協助評估人力仲介加盟的財務可行性。

## 線上瀏覽

**[https://jacksonyang0315.github.io/johnathan-franchise-analysis/](https://jacksonyang0315.github.io/johnathan-franchise-analysis/)**

## 功能

- **市場背景分析** — 台灣高齡化趨勢、看護需求數據、政策利多（含 Chart.js 互動圖表）
- **費用結構拆解** — 加盟金、權利金、文件費抵扣機制圖解
- **單案損益試算** — 完整 3 年聘期收入與成本明細
- **現金流模擬器** — 可自訂每月談成件數（1–8 件），即時計算回本時間與累計損益
- **風險評估清單** — HIGH / MED / TBD 分級，涵蓋財務、營運、法律面
- **行動 Checklist** — 可點擊勾選的決策前待辦事項

## 技術

- 純 HTML + CSS + JavaScript（單一檔案，無需建置）
- [Chart.js 4.4.7](https://www.chartjs.org/) 資料視覺化
- [IBM Plex Sans](https://fonts.google.com/specimen/IBM+Plex+Sans) + [Noto Sans TC](https://fonts.google.com/noto/specimen/Noto+Sans+TC) 字型
- 響應式設計，支援桌面與行動裝置
- `prefers-reduced-motion` 無障礙支援

本報告僅供內部決策參考，不構成投資建議。所有數據來源為加盟展簡報與現場筆記，未經獨立查證。
